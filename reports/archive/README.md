# Reports Archive

This directory contains archived audit reports and session summaries organized by month.

## Directory Structure

```
/reports/archive/
├── 2026-03/
│   ├── README.md
│   └── [March 2026 archived reports]
├── 2026-02/
└── 2026-01/
```

## Archive Policy

**Reports are archived when:**
- Report is 30+ days old
- Report content is superseded by newer audit
- Report findings have been addressed
- Report is no longer actively referenced

**Protected Reports (Never Archive):**
- Current month reports
- Master reference documents
- Active audit reports with pending tasks

## Retrieval

To find an archived report:
1. Check the month directory based on report date
2. Each directory has a README with report inventory
3. Use git history to track report lifecycle
